This AI prompt generates a hyper-realistic photo of a serious billiards player leaning over a vibrant blue pool table, focusing on a precise shot.

Generated Images

The Prompt 

Ultra-realistic medium close-up of a me (same face as reference image 100% accuracy), dark hair, focused serious gaze.Outfit: browm polo athletic jersey with TECH patch on left sleeve, number 07 on right, black smartwatch. Pose: leaning over vibrant blue billiard table, left hand forming bridge, cue stick aimed at white cue ball with red dot, red object ball visible.

How to Use This Prompt

1. Copy the prompt text. 2. Open your AI image tool like Midjourney, DALL-E, or Stable Diffusion. 3. If you have a reference photo of a face, upload it as instructed by your tool. 4. Paste the prompt. 5. Generate the image. Use an upscaler for higher detail.

Why This Prompt Works

This prompt works because it gives clear, photographic details. 'Ultra-realistic medium close-up' sets the style and framing. Specifying the outfit, pose, and key objects (cue stick, colored balls) gives the AI concrete elements to render. Mentioning '100% accuracy' for a reference face guides AI tools that support image prompting to match facial features closely, leading to a consistent and believable character portrait.

Prompt Variations

  • Change the jersey color to black with a gold number for a different team look.
  • Set the scene in a dark, moody pool hall with dramatic overhead lighting.
  • Make the player a woman with a determined expression.
  • Show the moment of impact, with the cue ball striking the red ball.
  • Change the sport to darts or chess for a different focused activity.

Frequently Asked Questions

  • What does 'medium close-up' mean in an AI prompt?
    It means the photo is framed from the chest or shoulders up, focusing on the subject's face and upper body while still showing some of the surroundings, like the pool table.
  • Can I use this prompt without a reference image?
    Yes. The AI will generate a random person with dark hair and a serious expression that fits the rest of the description.
  • Why is the cue ball described with a red dot?
    This specifies a 'cue ball with a red circle logo', a common detail on real pool balls. It helps the AI create a more accurate and recognizable billiards scene.
  • Which AI tool is best for this type of character portrait?
    Midjourney and DALL-E 3 are very good at understanding detailed clothing and pose descriptions to create realistic human portraits.
