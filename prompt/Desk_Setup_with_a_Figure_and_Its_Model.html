How to use this prompt

To use this prompt professionally, apply it for figure collectibles, 3D modeling demos, or toy packaging in hobbies and streaming. Preserve "highly detailed 1/7 scale commercialized figure," "couple character," "modern gaming/streaming setup," and "BANDAI-style toy packaging" for details. Do not change the acrylic base or ZBrush screen. Optionally vary pose; for example, static could yield display visuals, leading to webcam frames, while dynamic keeps branded, realistic outcomes.

Prompt

“Create a highly detailed 1/7 scale commercialized figure of the couple character from the illustration, rendered in a realistic style. The character is posed dynamically. The figure is placed on the broad desk of a modern gaming/streaming setup, next to a large microphone and mechanical keyboard in a condo-type bedroom. It uses a circular transparent acrylic base without any text. On the main ultrawide monitor,a live ZBrush session is visible, showing the 3D model of the very same figure. The BANDAI-style toy packaging box is positioned on the desk like a piece of branded merch with the photo reference, perfectly framed in the stream's webcam view. Focus on getting all the details right.”

Prompt Variations

  • Setup Style: Swap “modern gaming/streaming setup” → “a traditional artist’s workshop” with physical tools. Swap “condo-type bedroom” → “a professional studio” .
  • Software: Swap “ZBrush session” → “a Blender 3D viewport” for different software.
  • Packaging: Swap “BANDAI-style toy packaging” → “a Funko Pop! style box” for a different packaging aesthetic.
  • Figure Pose: Swap “posed dynamically” → “posed in a neutral stance” to better match the 3D model reference.
