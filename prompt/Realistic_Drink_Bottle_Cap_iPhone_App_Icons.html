The Prompt 

Create a set of 9 modern iPhone app icons designed as colorful drink bottle caps in a realistic 3D mockup style. Each cap should be viewed from above, with realistic shadows, slight metallic texture, and embossed ridged edges like a real bottle cap.Use bold, flat graphic logos in the center and curved app names around the rim, similar to vintage soda cap branding. Each cap should have a unique color scheme that reflects the app’s identity.

Apps to include:

Apple MusicStyle: photorealistic 3D rendering, high detail, soft studio lighting, slight reflection on paint, embossed ridges on edges, text and logos printed on cap surface.
Background: clean neutral surface, subtle shadow under each cap for depth.
Mood: fun, collectible, premium soda-cap aesthetic.

How to Use This Prompt

To use this prompt professionally, utilize it in app icon design, UI/UX portfolios, or branding for tech companies and app stores. Feed it into AI designers by listing specific apps like "Instagram" or "Spotify," preserving "modern iPhone app icons," "colorful drink bottle caps," "realistic 3D mockup," and "photorealistic 3D rendering" for a collectible aesthetic. Do not alter the 9-icon set, soft studio lighting, or embossed ridges. For instance, changing colors for "TikTok" to neon pinks could create vibrant social media icons, leading to fun, premium looks, while "Gmail" in reds might yield professional email visuals with metallic textured results.

Why This Prompt Works

This prompt will work because it cleverly combines nostalgia with modern app icon design by reimagining icons as vintage soda bottle caps it creates a fun collectible aesthetic that stands out the specifications for realistic 3D mockups with metallic textures embossed edges and bold graphics ensure a high-quality photorealistic result the unique color scheme for each app aids in immediate recognition while maintaining a cohesive set the clean neutral background with subtle shadows keeps the focus on the icons making them pop this concept is perfect for branding themes startup launches or anyone looking to give their phone interface a unique and playful look.

Prompt Variations

  • App Selection: Swap “Instagram TikTok Spotify YouTube Twitter/X WhatsApp Pinterest Gmail Apple Music” → “a set of gaming apps: Discord, Twitch, Steam, Xbox, etc.”. Swap for “a set of productivity apps: Slack, Notion, Zoom, Google Drive, etc.”.
  • Cap Style: Swap “vintage soda cap branding” → “modern craft beer cap branding” with simpler logos. Swap “embossed ridged edges” → “a smooth, crown-style cap” for a different design.
  • Color Scheme: Directive: “Each cap should have a matte pastel color scheme” for a softer, modern look. Directive: “Each cap should be a metallic color (gold, silver, copper)” for a premium feel.
  • Surface: Swap “clean neutral surface” → “a weathered wooden table” for a rustic background. Swap “clean neutral surface” → “a reflective metallic surface” for dramatic reflections.
