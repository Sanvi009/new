How to use this prompt

To use this prompt professionally, incorporate it into makeup brand campaigns, fashion editorials, or theatrical posters for beauty and advertising industries. Apply it in image creators by keeping "luxury makeup brand advertisement," "four stylish pop-art inspired girls," "separate window frames," and "bold colorful makeup" intact for theatrical composition. Do not change the beige wall or exaggerated poses. Optionally vary items like "lipstick" to "eyeshadow"; for example, adding "giant compact" could result in playful ads, while "mascara wand" might create dramatic eye-focused visuals with expressive, high-energy outcomes.

Prompt

"A luxury makeup brand advertisement featuring four stylish pop-art inspired girls, each posed in separate window frames in a beige wall, theatrical composition, one holding a mirror, another holding a lipstick, another holding a mascara wand, one with exaggerated oversized makeup item like a giant lipstick or compact, each styled with bold colorful makeup and expressive facial poses

Why this prompt works

This prompt works by creating a theatrical and stylish advertisement for a luxury makeup brand the concept of four pop-art inspired girls in separate window frames each holding makeup items creates a dynamic and colorful composition the exaggerated oversized items and bold makeup add a fun dramatic twist that captures attention this setup is perfect for branding that wants to appear high-end yet playful and artistic the prompt’s details ensure a cohesive and vibrant image suitable for social media campaigns billboards or editorial spreads that aim to stand out.

Prompt Variations

  • Number of Subjects: Swap “four stylish pop-art inspired girls” → “two stylish pop-art inspired girls and two boys” for mixed gender.
  • Makeup Item Focus: Swap “a giant lipstick or compact” → “a giant eyeshadow palette” for a different focus. Swap “holding a mirror, another holding a lipstick, another holding a mascara wand” → “all holding different sized brushes”.
  • Wall Color: Swap “beige wall” → “a bright pink wall” for a more energetic backdrop. Swap “beige wall” → “a graffiti-covered brick wall” for an urban pop art feel.
  • Style: Swap “pop-art inspired” → “punk-rock inspired” with bold makeup and edgy hairstyles.
