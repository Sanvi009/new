Create an ultra-realistic 8K AI image of an Asian man sitting on a golden throne. He wears a sharp black suit with a red tie, holding a lit cigar with smoke rising. Around him stand six women in black dresses embroidered with red dragons, posing with confidence. The scene takes place in a dark Japanese-style room, lit with golden lights and a giant dragon painting on the wall. This AI prompt captures power, authority, and cinematic drama.

The Prompt 

Convert the image into a set of 15 chibi sticker (4x4 grid) with outfit similar to this one, including expression of laughing, being angry, crying, sulking, thinking, being sleepy, blowing a kiss, winking, being surprise, in awe, wanting a hug, in love, etc

How to Use This Prompt

To use this prompt professionally, apply it for sticker pack creation in digital apps, social media graphics, or merchandise like phone cases and planners. Implement it in tools like Adobe Illustrator AI by using an uploaded image as reference, replacing expressions with the listed ones but keeping "chibi sticker (4x4 grid)," "outfit similar to this one," and 15 variations unchanged. Do not modify the grid layout or anthropomorphic style. Changing expressions to include "excited" could yield energetic emoji sets for messaging apps, resulting in fun, emotive collections, whereas adding "confused" might produce relatable reaction stickers with diverse emotional outcomes.

Why This Prompt Works

This prompt is highly effective for creating engaging and versatile digital stickers it specifies a set of 15 chibi expressions in a 4x4 grid which is perfect for messaging apps social media or gaming the variety of emotions from laughing to crying ensures wide usability and appeal the instruction to include similar outfits maintains brand consistency if used for a character the chibi style is universally popular for its cuteness and expressiveness making these stickers highly shareable and marketable this prompt is ideal for artists looking to create sticker packs for digital platforms as it meets the demand for diverse and emotional expressive content.

Prompt Variations

  • Outfit Style: Swap “outfit similar to this one” → “a different casual outfit: t-shirt and jeans”. Swap “outfit similar to this one” → “a formal outfit: suit and tie”.
  • Outfit Theme: Swap “outfit similar to this one” → “a fantasy warrior outfit”. Swap “outfit similar to this one” → “a cute animal onesie”.
  • Grid Layout: Swap “4x4 grid” → “a 3x5 grid” for 15 stickers in a different arrangement.
  • Expression Set: Modify the list: swap “blowing a kiss, winking” → “eating, yelling excitedly” for a different set of actions.
