How to use this prompt

To use this prompt professionally, apply it for toy designs, collectible figurines, or character merchandising in animation and games. Preserve "3D glass figurine character," "transparent shiny body," "yellow shirt, mustard-yellow jacket," and "solid bright yellow background" for glossy look. Do not change the stylized face or vinyl toy style. Changing hat color to "red" could produce vibrant visuals, leading to friendly smiles, while brown keeps detailed, modern results.

Prompt

A 3D glass figurine character with a transparent shiny body, styled in a cute toy-like design. The figure wears a yellow shirt, a mustard-yellow jacket, brown belt, and a brown hat with a black band. It has a stylized cartoon face with black glasses, a brown beard, and a friendly smile. The background is a solid bright yellow, creating a clean and vibrant look. The overall style is modern, glossy, and highly detailed, resembling a collectible designer toy or vinyl figurine."

Why this prompt works

This prompt is highly effective for a cute and glossy 3D glass figurine with a friendly design the transparent shiny body and stylized cartoon features add charm and modernity the bright yellow background ensures the figure stands out making it perfect for collectible toys branding mascots or social media content that aims to be vibrant and engaging.

Prompt Variations

  • Character Design: Swap the described character → “a transparent glass figurine of a superhero”. Swap the described character → “a transparent glass figurine of an animal, like a fox”.
  • Clothing Palette: Swap “yellow shirt, mustard-yellow jacket, brown belt, and a brown hat” → “a blue shirt, silver jacket, and black accessories” for a different color scheme.
  • Background: Swap “solid bright yellow” → “a light blue gradient” for a different vibrant background. Swap “bright yellow” → “white” for a cleaner, more product-shot background.
  • Style: Swap “modern, glossy” → “vintage, frosted glass” for an older look. Swap “highly detailed” → “simplified, minimalist shapes” for a different aesthetic.
